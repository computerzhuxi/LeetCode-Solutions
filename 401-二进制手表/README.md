# 401. 二进制手表

## 题目描述

[401-二进制手表](https://leetcode.cn/problems/binary-watch/)

## 思路

### 方法一: 枚举时分

有题可知小时由4位的二进制表示，分钟由6位的二进制表示，灯亮为1，灯灭为0。
我们可以枚举所有时分，将该时钟的二进制中1的个数之和与 *turnedOn* 相比较，若相等则加入字符串列表中

### 方法二：二进制枚举

另一种枚举方法是枚举所有 2^10 =1024 种灯的开闭组合，即用一个二进制数表示灯的开闭，其高 4 位为小时，低 6 位为分钟。若小时和分钟的值均在合法范围内，且二进制中 1 的个数为 *turnedOn* ，则将其加入到答案中。

## 复杂度

- 时间复杂度：O(1)
- 空间复杂度：O(1)
