# 76. 最小覆盖子串

## 题目描述

[76-最小覆盖子串](https://leetcode.cn/problems/minimum-window-substring/)

## 思路

用一个 map 或者用数组存储 t 中的字符出现的次数，然后用两个指针 left 和 right，left 指向窗口的起始位置，right 指向窗口的末尾位置。

先通过遍历的方法找到一个符合条件的窗口，并记下当前窗口的起点和长度。

移动左边界，直到窗口内的存在的 t 中的字符的数量小于 t 中对应的字符，然后再移动右边界，直到窗口内的存在的 t 中的字符的数量大于等于 t 中对应的字符。看看当前窗口的起点和长度是否比之前的窗口的起点和长度更小，如果是，则更新答案。

重复上述过程，直到 right 指针到达字符串的结尾。

## 复杂度

- 时间复杂度：O(|s| + |t|)
- 空间复杂度：O(C) t 中的字符集大小
