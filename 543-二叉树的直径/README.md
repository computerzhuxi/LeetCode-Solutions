# 543. 二叉树的直径

## 题目描述

[543. 二叉树的直径](https://leetcode.cn/problems/diameter-of-binary-tree/)

## 思路

二叉树的容易两个节点的最长路径有两种情况一种是经过根节点，一种是不经过

经过根节点的最长路径为两个子树的深度之和

不经过根节点的路径我们可以以递归的方式来变成第一种情况

所以我们可以写一个递归函数，它返回的是该根节点的树的深度。通过递归得到两个子树的深度，算出该根节点下的最长路径。我们用一个变量存储最长路径，并用这个变量和我们算好的最长路径比较。最后返回深度。

## 复杂度

- 时间复杂度：O(n)
- 空间复杂度：O(height)
