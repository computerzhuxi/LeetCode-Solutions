# 34. 在排序数组中查找元素的第一个和最后一个位置

## 题目描述

[34-在排序数组中查找元素的第一个和最后一个位置](https://leetcode.cn/problems/find-first-and-last-position-of-element-in-sorted-array/)

## 思路

根据题意，可以发现，我们要找到第一个和最后一个为目标值的数。要求我们使用二分查找的方法来寻找这两个位置。我们可以通过查找第一个不小于目标值的位置和第一个大于目标值的位置来做

第一个位置我们只需要在查找的同时，用一个数记录查找位置不小于目标值的位置。当查找结束时，这个记录就是第一个不小于目标值的位置。

第二个位置我们可以在查找的同时，用一个数记录查找位置大于目标值的位置，但查找结束时，这个记录就是第一个大于目标值的位置。

我们可以通过这两个位置知道查找元素的第一个和最后一个位置

## 复杂度

- 时间复杂度：O(log n)
- 空间复杂度：O(1)
