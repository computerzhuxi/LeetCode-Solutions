# 438. 找到字符串中所有字母异位词

## 题目描述

[438-找到字符串中所有字母异位词](https://leetcode.cn/problems/find-all-anagrams-in-a-string/)

## 思路

一个很简单的方法，就是遍历字符串 `p` 中的每个字符出现的次数，然后在字符串 `s` 取和字符串 `p` 相同长度的子串，如果两者的每次字符出现的次数一样的话，即为字符串 `p` 的一个异位词。

像这样每次都要比较一次两者的字符出现的次数，有太多没有必要的比较了，我们可以不用分别存储两者的字符出现的次数，而是用一个数组来存储字符串`p` 和子串中每次字符出现的次数差值。字符串 `p` 中出现一次就减 1，子串出现一次就加 1，同时使用一个变量`differ`记录当前窗口的子串和字符串 `p` 的字符不同的个数。每次移动窗口，只需要更新移出和移入的字符的差值，并同步更新`differ`的值，如果`differ`为 0，则找到了一个异位词。

## 复杂度

- 时间复杂度：O(n+m+Σ)， 其中 n 为字符串 s 的长度，m 为字符串 p 的长度，其中 Σ 为所有可能的字符数。我们需要 O(m) 来统计字符串 p 中每种字母的数量；需要 O(m) 来初始化滑动窗口；需要 O(Σ) 来初始化 differ；需要 O(n−m) 来滑动窗口并判断窗口内每种字母的数量是否与字符串 p 中每种字母的数量相同，每次判断需要 O(1) 。因为 s 和 p 仅包含小写字母，所以 Σ=26。

- 空间复杂度：O(Σ)。用于存储滑动窗口和字符串 p 中每种字母数量的差。
