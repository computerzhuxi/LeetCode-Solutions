# 207. 课程表

## 题目描述

[207-课程表](https://leetcode.cn/problems/course-schedule/)

## 思路

用 DFS 遍历图，判断是否有环：

每个节点有三种状态：

0：未访问

1：正在访问（在当前 DFS 路径上）

2：已访问完成（没有环）

如果在 DFS 过程中遇到状态为 1 的节点，说明有环，无法完成。

步骤：

1. 建邻接表 graph，graph[b] 存放所有依赖 b 的课程（即 b → a）。

2. 对每个未访问的节点做 DFS：

    - 标记为 1

    - 遍历它的所有后继，如果后继状态为 1 → 有环

    - 如果后继未访问，递归 DFS

   - 递归返回 False 则直接返回 False

3. 遍历完标记为 2

4. 所有节点都无环则返回 True

## 复杂度

- 时间复杂度：O(n+m)
- 空间复杂度：O(n+m)
