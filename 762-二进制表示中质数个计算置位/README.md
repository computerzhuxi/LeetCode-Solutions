# 762. 二进制表示中质数个计算置位

## 题目描述

[762-二进制表示中质数个计算置位](https://leetcode.cn/problems/prime-number-of-set-bits-in-binary-representation/)

## 思路

我们可以发现 right<10^6<2^20，因此，二进制中1的个数不会超过20。我们可以一一列举出来：2，3，5，7，11，13，17，19。
我们可以用一个二进制数 mask = 665772 = 10100010100010101100 来存储这些数。从低位到高位的1表示这个数为质数。
遍历所有数，得到他们的1的个数为x，令2^x和mask按位与，如果不为零，就是质数。

## 复杂度

- 时间复杂度：O(n)
- 空间复杂度：O(n)
