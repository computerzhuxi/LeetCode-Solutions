# 3666. 使二进制字符串全为 1 的最少操作次数

## 题目描述

[3666-使二进制字符串全为 1 的最少操作次数](https://leetcode.cn/problems/minimum-operations-to-equalize-binary-string/)

## 思路

遍历所有情况
设c为我们当前操作取的'0'的个数
每次操作最多取min(m,k)个'0'
0 <= c <= min(m,k)
每次操作最多取n-m个'1'
k - c <= n - m 即 k - n + m <= c
我们就可以得到c的范围
k - n + m <= c <= min(m,k)
操作后的m的为
m + k - 2 \* c
我们可以很明显的看出来，这个操作后的m的奇偶性在这个范围类都是一样的，它们只和 m+k 有关。
我们就可以用两个set分别存储奇数偶数的m情况，并用数组dist存储每种m情况下所需要的操作数，使用广度优先，每次都求出c的范围，在遍历这个范围的所有m情况，将所有情况都加入队列里面并将上一步的m情况下的步数加1存储到对应的dist中。
最后我们会得到所有可到达情况下的步数，我们只需要看dist[0]里存储的数字就能得到答案。

## 复杂度

- 时间复杂度：O(n log n)
- 空间复杂度：O(n)
