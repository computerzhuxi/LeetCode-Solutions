# 51. N 皇后

## 题目描述

[51-N 皇后](https://leetcode.cn/problems/n-queens/)

## 思路

这个也可以用回溯的方法来做，皇后不能在同一行或同一列或同一斜线上，那么我们可以知道每行只能摆放一个皇后，那么我们就遍历每行皇后能摆放的位置。

另外三个条件我们可以使用三个集合来表示该位置是否能放置。但是这样的话又需要三个数组空间了。我们发现n是小于等于9的。那么我们是不是可以使用位运算来做

我们用三个数分别代表列和两个斜线。对于每一行：
- 列：我们只需要计算当前行放置的位置和之前的代表行的数相与即可求得下一行关于列的位置限制
- 斜线：我们可以发现下一行的限制是在当前行的位置分别向左移位和像右移位得到的。那么，我们就可以直接将表示两斜线的两个数分别与当前行位置的数相与，然后再分别向左向右移位即可

## 复杂度

- 时间复杂度：O(n!)
- 空间复杂度：O(n)
