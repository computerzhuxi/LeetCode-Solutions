# 1. 两数之和

## 题目描述

[1-两数之和](https://leetcode.cn/problems/two-sum/)

## 思路

一个很明显的想法是暴力枚举，对数组中的每一个数都和其他数相加看是不是等于目标值，如果是，则返回这两个数的索引。

这种方法的时间复杂度是 O(n^2)，空间复杂度是 O(1)。

更好的方法是使用哈希表，对数组中的每一个数都进行一次查找，看是否存在另一个数使得两数之和等于目标值。如果有则返回这两个数的索引。如果没有则将这个数存储在哈希表中，key 为这个数的值，value 为它的索引。

这种方法的时间复杂度是 O(n)，空间复杂度是 O(n)。

## 复杂度

- 时间复杂度：O(n)
- 空间复杂度：O(n)
