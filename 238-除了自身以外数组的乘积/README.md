# 238. 除了自身以外数组的乘积

## 题目描述

[238. 除了自身以外数组的乘积](https://leetcode.cn/problems/product-of-array-except-self/)

## 思路

要求不用除法来做，我们观察每个位置的的值都是前面的积乘后面的积，那么我们可以从左往右遍历数组，记录每个位置的左侧乘积，然后从右往左遍历数组，记录每个位置的右侧乘积，最后再乘起来即可。

但是这样还要两个额外的数组去存储前缀的积和后缀的积。

我们可以发现，从左往右遍历实际上我们前缀或后缀的积只用到了一次，且 `i` 位置的前缀积是 `i-1` 位置的前缀积乘 `i-1` 元素。那么我们是不是可以只用一个数来表示前缀积呢？这样我们就不需要再用一个数组来存储前缀积了。后缀值同理。这样我们只需要用两个变量来记录前缀积和后缀积即可。

## 复杂度

- 时间复杂度：O(n)
- 空间复杂度：O(1)
