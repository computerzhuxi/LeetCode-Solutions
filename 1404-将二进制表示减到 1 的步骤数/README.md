# 1404. 将二进制表示减到 1 的步骤数

## 题目描述

[1404-将二进制表示减到 1 的步骤数](https://leetcode.cn/problems/number-of-steps-to-reduce-a-number-in-binary-representation-to-one/)

## 思路

我们很容易发现，当低位为1时，需要2步操作，低位为0时，只需要1步。
但是，当低位为1时，它前面的数又会是什么情况呢？
当它为1时，它会加1，导致其变为1，就只需要1步操作了
当它为0时，它会变成1，就跟低位为1时的操作一样，需要两步了。
同时，也要注意当二进制只剩1位时，其必然为1，如果前面加1了，会导致其进行一次除2操作，如果没有的话，则不需要操作。

## 复杂度

- 时间复杂度：O(n)
- 空间复杂度：O(1)
