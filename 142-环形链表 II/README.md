# 142. 环形链表 II

## 题目描述

[142-环形链表 II](https://leetcode.cn/problems/two-sum/)

## 思路

这题我们依然可以用哈希表。但是我想用快慢指针来做

我们将这个链表分为三个部分，设链表中环外部分的长度为 $a$ 。 `slow` 指针进入环后，又走了 $b$ 的距离与 `fast` 相遇，环剩下的部分长度为 $c$ 。此时，`fast` 指针已经走完了环的 `n` 圈，因此它走过的总距离为 $a+n(b+c)+b=a+(n+1)b+nc$ 。

任意时刻， `fast` 指针走过的距离是 `slow` 指针的 $2$ 倍。因此，我们有
$$
a+(n+1)b+nc=2(a+b)+m(b+c) \rightarrow a=(n-m-1)(b+c)+c
$$

此时， `slow` 指针也走了 $m$ 圈， `fast` 指针必定要比 `slow` 指针至少多走一圈，所以 $n-m-1 > 0$

所以我们在找到相遇点之后，只需要让另一个指针指向头指针，然后这个指针和 `slow` 一起移动，最后它们会在入环点相遇。

## 复杂度

- 时间复杂度：O(n)
- 空间复杂度：O(1)
