# 208. 实现 Trie (前缀树)

## 题目描述

[208-实现 Trie (前缀树)](https://leetcode.cn/problems/implement-trie-prefix-tree/)

## 思路

前缀树就是有一颗树，从根到叶子节点的路径为该字符串，其中从根出发的部分路径为前缀。我们搜索前缀只需从根出发，看看它的子节点是否存在对应字符。插入则是如果没有就新建一个节点插入。

因为是英文字母且都是小写，我们在创建节点的时候可以只用将数组长度设置为26个，查找字符的时候将字符减去'a'的数值就可以很快查找到对应字符的位置。

## 复杂度

- 时间复杂度：O(1)
- 空间复杂度：O(∣T∣⋅Σ)，其中 ∣T∣ 为所有插入字符串的长度之和，Σ 为字符集的大小，本题 Σ=26。
