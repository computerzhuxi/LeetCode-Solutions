# 994. 腐烂的橘子

## 题目描述

[994-腐烂的橘子](https://leetcode.cn/problems/rotting-oranges/)

## 思路

广度优先搜索，先遍历一遍，将橘子的数量存下来，并使用另一个矩阵来存储橘子是在什么时候腐烂的。

每次遍历都将该腐烂橘子的周围新鲜橘子腐烂，并将被腐烂的新鲜橘子的腐烂时间为当前橘子的加一。并记下腐烂时间。

因为队列里面新加入的腐烂橘子一定比前面的晚，所以遍历完的记下的腐烂时间就是答案

遍历完后，如果新鲜橘子大于零，返回-1，否则返回记录的腐烂时间。

## 复杂度

- 时间复杂度：O(nm)
- 空间复杂度：O(nm)
