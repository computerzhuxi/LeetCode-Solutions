# 160. 相交链表

## 题目描述

[160-相交链表](https://leetcode.cn/problems/intersection-of-two-linked-lists/)

## 思路

1. 计算长度

    分别遍历两个链表，统计出链表 `A` 的长度 $lenA$ 和链表 `B` 的长度 `lenB`

2. 对其起点

    计算两个链表的长度差值 `diff` ，让较长的链表先从头节点出发，走 `diff` 步。此时，两个链表的指针就处于同一水平起点（剩余未遍历的长度相同）
    
3. 同步遍历找交点

    让两个指针同时向后移动，每次各走一步。当两个指针指向同一个节点时，这个节点就是两个链表的相交节点；如果遍历到链表末尾都没有相遇，说明两个链表不相交。
## 复杂度

- 时间复杂度：O(n + m)
- 空间复杂度：O(1)
