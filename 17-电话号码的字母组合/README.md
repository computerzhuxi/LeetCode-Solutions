# 17. 电话号码的字母组合

## 题目描述

[17-电话号码的字母组合](https://leetcode.cn/problems/letter-combinations-of-a-phone-number/)

## 思路

这题是给定一个包含数字的字符串，返回它能表示的所有字母组合。可以用回溯解决。

- 我们可以使用哈希表来存储每个数字代表的字母串
- 从第一个数字出发，依次选择一个字母，然后递归下一个数字。
- 当选择的字母个数等于数字字符串的长度时，就得到了一个完整的组合。

## 复杂度

- 时间复杂度：O( $3^m \times 4^n$ )
- 空间复杂度：O(m + n)
