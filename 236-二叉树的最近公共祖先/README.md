# 236. 二叉树的最近公共祖先

## 题目描述

[236-二叉树的最近公共祖先](https://leetcode.cn/problems/lowest-common-ancestor-of-a-binary-tree/)

## 思路

有两种方法。

一种是将所有的节点和它的父节点用哈希表存储下来，然后然后在分别访问两节点的父节点。这样第一个重复访问的节点就是两节点的公共祖先

一种是搜索二叉树，符合一下两个条件中的一个的节点既是我们要找的节点：
    1. 左右子树都找到了目标节点
    2. 左右子树有一个找到了目标节点，该节点也是目标节点

## 复杂度

- 时间复杂度：O(n)
- 空间复杂度：O(n)
