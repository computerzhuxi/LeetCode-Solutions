# 189. 轮转数组

## 题目描述

[189-轮转数组](https://leetcode.cn/problems/two-sum/)

## 思路

有三个方法，第一种就是用另一个数组缓存结果，这个方法需要额外数组的空间。空间复杂度为 O(n)。

第二个方法是从第一个开始，找到它要到的位置，然后交换元素，继续找到下一个位置交换元素，直到到达最开始的位置。和快排的方法很类似，总共要跑 n 和 k 的最小公约数个次数。

第三个方法是先将整个数组反转，然后再反转第一个到第 k mod n 个元素，再反转剩下的元素。

三种方法除了第一个方法的空间复杂度为 O(n) 外，其他两个方法的空间复杂度都为 O(1)。时间复杂度都为 O(n)。

## 复杂度

- 时间复杂度：O(n)
- 空间复杂度：O(1)
