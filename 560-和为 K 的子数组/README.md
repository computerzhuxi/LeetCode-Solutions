# 560. 和为 K 的子数组

## 题目描述

[560-和为 K 的子数组](https://leetcode.cn/problems/subarray-sum-equals-k/)

## 思路

我们定义 $pre[i]$ 为 $[0,...,i]$ 里所有数的和。那么 $pre[i]$ 与 $pre[j]$ 的差值 $pre[i]-pre[j - 1]$ 就是 $[j,i]$ 里所有数的和。

我们可以用哈希表存储 $pre[j - 1]$ 的数量，每次查找 $pre[i]-k$ 是否在哈希表中，如果存在，则说明 $[0,i]$ 里存在 $k$ 的子数组。

我们可以看出，$j$ 一定在 $i$ 之前，而 $pre[i]=pre[i-1] + nums[i]$ ，所以我们不需要存储所有的 $pre$ 值。遍历数组时，通过 $pre$ 和 $nums[i]
$ 计算当前的 $pre$ 值，并更新哈希表。

## 复杂度

- 时间复杂度：O(n)
- 空间复杂度：O(n)
