# 23. 合并 K 个升序链表

## 题目描述

[23-合并 K 个升序链表](https://leetcode.cn/problems/merge-k-sorted-lists/)

## 思路

我们可以采用堆排序的方法，将所有链表的第一个元素放入堆里面排序。每次取走一个时，将这个链表的下一个元素放入堆中。

C++中的 `priority_queue` 就是使用堆排序实现的，它会根据我们定义的规则来排序。这样我们可以写一个结构，将链表当前元素的值和地址存进去，再设定 `<` 的规则。我们就可以将所有链表以这个结构体的形势存储下来，放到 `priority_queue` 中，它会自动完成排序。我们只需要每次将队首元素拿出来，将对应链表的下一个元素存储进去即可完成合并。

当然，我们也可以采用归并的方法来合并。

## 复杂度

- 时间复杂度：O(kn x log k)
- 空间复杂度：O(k)
