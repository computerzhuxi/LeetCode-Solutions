# 35. 搜索插入位置

## 题目描述

[35-搜索插入位置](https://leetcode.cn/problems/search-insert-position/)

## 思路

因为数组已经排好序，所以用二分查找。

1. 设置左右指针，在数组范围内不断折半查找。、

2. 每次取中间位置：

- 如果中间值等于目标值，直接返回这个索引。
- 如果中间值小于目标值，说明目标在右半部分，左指针移到中间右边。
- 如果中间值大于目标值，说明目标在左半部分，右指针移到中间左边。

3. 如果循环结束都没找到目标值，此时左指针指向的位置就是目标值应该插入的地方（第一个大于目标值的位置）。

## 复杂度

- 时间复杂度：O(log n)
- 空间复杂度：O(1)
