# 693.交替位二进制

## 题目描述

[693-交替位二进制](https://leetcode.cn/problems/binary-number-with-alternating-bits/)

## 思路

### 方法一：模拟

看二进制是否0、1交替，可以每次看两位，所以，我们可以每个循环将该数 mod 4，这样得出来的就是后两位数的十进制数，很明显1和2是交替的，0和3是不交替的

#### 复杂度

- 时间复杂度：O(log n)
- 空间复杂度：O(1)

### 方法二：位运算

我们可以将这个数向右移一位，然后再让它与原来的数按位异或，这样每个数都会和与前一位进行异或操作（除了最左边的那位是和0异或）。这样，如果是交替的，结果一定是11……1，这样的数。那么我们可以让这个数加1，和原数按位与，得到的结果就一定为0。如果不是交替的话这样操作的结果就一定不是0，很容易得到结果。

#### 复杂度

- 时间复杂度：O(1)
- 空间复杂度：O(1)
