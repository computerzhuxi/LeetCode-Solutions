# 73. 矩阵置零

## 题目描述

[73-矩阵置零](https://leetcode.cn/problems/set-matrix-zeroes/)

## 思路

如果我们不考虑空间复杂度，我们可以用两个数组分别表示行列是否需要置零。这个时候空间复杂度为 O(mn)。

但是题目要求空间复杂度为 O(1)，有没有什么方法可以不用这两个数组呢？

我们可以将矩阵的第一列和第一行代替这两个数组工作，对于需要置零的元素，我们将其所在的行列的第一列和第一行置为 0。我们可以用两个变量来记录第一列和第一行是否需要置零。

这样空间复杂度就降低到了 O(1)。但是我们发现第一行第一列这个元素被浪费了，我们可以用一个变量来记录第一列是否需要置零，第一列的元素来记录哪一行需要置零，而第一行从第二个元素开始记录哪一列需要置零

## 复杂度

- 时间复杂度：O(mn)
- 空间复杂度：O(1)
